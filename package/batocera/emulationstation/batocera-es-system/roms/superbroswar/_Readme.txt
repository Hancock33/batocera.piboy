This port requires a copy of the Super Mario War data files.
Download https://github.com/mmatyas/supermariowar/releases/download/continuous/supermariowar_2022-01-08_linux.zip and extract to contents of the data directory to this directory

This folder should look like ...

filters
gfx
maps
music
scripts
sfx
superbroswar.game
tours
worlds
_Readme.txt
_info.txt
